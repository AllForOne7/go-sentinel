  Sentinel Agent — установка на удалённый компьютер
═══════════════════════════════════════════════════════

СОДЕРЖИМОЕ ПАПКИ:
  agent.exe   — агент мониторинга (Windows x64)
  .env        — файл настроек
  start.bat   — скрипт запуска

УСТАНОВКА:
  1. Скопируй всю папку на целевой компьютер
  2. Открой файл .env в блокноте
  3. Замени СЕРВЕР_IP на IP-адрес основного сервера Sentinel
     Пример: NATS_URL=nats://192.168.1.100:4222
  4. Измени HOST_NAME на понятное имя компьютера
     Пример: HOST_NAME=office-pc-2
  5. Запусти start.bat

ВАЖНО:
  - На основном сервере должен быть запущен NATS (порт 4222)
  - Порт 4222 должен быть открыт в файрволе основного сервера
  - Агент отправляет метрики каждые 5 секунд
  - Speedtest запускается каждые 30 минут (настраивается в .env)
  - Логи сохраняются в папку logs/

АВТОЗАПУСК (Windows):
  1. Нажми Win+R, введи shell:startup
  2. Создай ярлык на start.bat в открывшейся папке
  3. Агент будет запускаться при входе в систему
